# Comptes rendus de réunion

Ce dossier centralise les comptes rendus des réunions formalisées du projet R4, conformément à la comitologie décrite dans [`MEETINGS.md`](../../MEETINGS.md).

## Convention de nommage

`YYYY-MM-DD-MX.md` — où `YYYY-MM-DD` est la date effective de la réunion et `MX` l'identifiant de la réunion (M1 à M6).

Exemples :
- `2026-04-28-M1.md` — kickoff périmètre formations
- `2026-05-12-M6-1.md` — sync hebdo MA, première session

## Template

Voir la section 5 de [`MEETINGS.md`](../../MEETINGS.md) pour le format normé en quatre rubriques (avancement, points bloquants, décisions, prochaines étapes).
